# Admin Command Box

The **Admin Command Box** is a lightweight, Python-based productivity tool designed to bridge the gap between complex terminal operations and user-friendly administration.

## Why I Built This
As a developer, I noticed that even the most powerful console commands are often forgotten or feel intimidating to those who don't spend their entire day in a terminal. I built this app to provide a "digital safety net"â€”a centralized, searchable place for essential commands (like Git, TrueNAS management, or SIP Trunk configurations). 

The goal was to create a tool that offers:
* **Efficiency:** No more searching through browser bookmarks or old notes.
* **Security:** Execute pre-verified commands without the risk of typos.
* **Empowerment:** Helping less experienced console users feel confident in a CLI environment.

## Features & Benefits
* **Categorized Command Storage:** Organize your knowledge into logical groups (e.g., GIT, System, Networking).
* **SQLite Backend:** A robust, local database ensures your data stays private and fast.
* **Simplicity First:** Designed for clarity and ease of use.

## How to Use It
1. **Launch:** Run the main script via `python3 main.py`.
2. **Search:** Enter a category or a keyword to find the command you need.
3. **Learn/Execute:** View the command along with its description to understand exactly what it does before you run it.
